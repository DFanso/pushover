Pushover POC

This proof of concept sends an ecommerce-style notification through the Pushover Message API using your app token and user/group key from .env.

  • Create a Pushover app token at https://pushover.net/apps/build.
  • Use your dashboard user key, or a group key, as PUSHOVER_USER_KEY.
  • Normal ecommerce events should use priority 0; reserve higher priority for urgent cases.
  • If iPhone sound is missing, try an explicit sound like pushover or cashregister.
